Add VPC routes

```text
cosmic-cli vpc route add CIDR[,CIDR,CIDR] via NEXTHOP [flags]
```

## Options

```text
  -h, --help              help for add
  -p, --profile string    specify profile(s) to use
      --vpc-id string     specify VPC id
      --vpc-name string   specify VPC name
```
