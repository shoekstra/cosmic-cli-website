Delete VPC routes

```text
cosmic-cli vpc route delete [ cidr=CIDR | nexthop=NEXTHOP ] [flags]
```

## Options

```text
  -h, --help              help for delete
  -p, --profile string    specify profile(s) to use
      --vpc-id string     specify VPC id
      --vpc-name string   specify VPC name
```
